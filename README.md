# Movie Database Backend

This project is a Python backend module that connects to a MySQL database and performs queries 

It was developed as part of a university assignment and focuses on database interaction, SQL queries, and backend logic using Python.

## Technologies Used

- Python
- MySQL
- PyMySQL

## Features

- Update movie ratings based on user input
- Find colleagues of colleagues between actors
- Retrieve actor pairs based on movie genres
- Select top N actors per genre
- Trace direct and indirect actor influence through movie genres and years
